# Penjelasan Program

## Chaining Then-Catch
pertama buat function yang memuat 2 parameter,
kedua buat didalamnya function callback dengan 2 parameter resolve dan reject yang menjalankan sytax async  yang akan mengembalikan output reject dan resolve

handle fungsi tersebut dengan cara memanggil fungsi tersebut dengan berisikan 2 parameter
 
## Async-Await
pertama buat function yang memuat 2 parameter,
kedua buat didalamnya function callback dengan 2 parameter resolve dan reject yang menjalankan sytax async  yang akan mengembalikan output reject dan resolve

ketiga buat function baru dengan 3 parameter yang akan menjadi output
keempat handle dengan try sebagai program menangkap response resolve dengan menambahkan await sebelum memanggil function yang berisikan 2 parameter
kelima hendlele dengan catch untuk menangkapn response reject
